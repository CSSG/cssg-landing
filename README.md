# CSSG landing page

Static site built with [assemble.io](http://assemble.io/).


## Install

Based on [Node](https://nodejs.org), requires grunt, so:
```
npm i -g grunt grunt-cli
```


## Build

First launch (checkout CSSG-dependent repos):
```
grunt build --fetch
```

Following launches:
```
grunt build
```


## Development

```
grunt dev
```
